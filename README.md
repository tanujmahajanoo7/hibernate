# Hibernate Project Setup Guide

## Step-by-Step Instructions

### 1. Create New Project
- Select **New**
- Choose **Other**
- Select **Maven Project**
  - Search for: `maven-archetype-quickstart`
  - Use the quickstart archetype

### 2. Create Hibernate Configuration File
- Create file: `hibernate.cfg.xml`

### 3. Add Dependencies
Add the following dependencies from [Maven Repository](https://mvnrepository.com/):
- Hibernate
- JavaX (Java Persistence API)
- MySQL Connector

---

**Note**: Dependencies should be added to your `pom.xml` file.
